This application is an image filter incorporating small amongi in any picture. The resulting image can be saved by right-cliking on it and downloading the content. If you cannot see amongi, either the filter failed or the size was too small. So either download and zoom in the picture, or use the filter with bigger amongi.







You can go to Github if you want to improve the project, for example by improving the amongus placing algorithm.

https://github.com/TheLortex/amogu
